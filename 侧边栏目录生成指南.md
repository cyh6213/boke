# 侧边栏目录生成指南

给本项目任意页面添加侧边栏目录的标准流程。

## 步骤

### 1. 确定页面有哪些章节

每个章节对应一个顶级锚点，子章节对应次级锚点。例如 Workflow 项目页的层级：

```
顶级：Workflow 1.0（前身）、项目来源、核心亮点、其他亮点、测评结果
子级（归属核心亮点）：KAHN 算法、节点级回滚、流程可视化
子级（归属其他亮点）：Sub Agent 执行、三级异常处理
```

### 2. 给章节容器加 ID

- **顶级章节**：在 `.project-card` 上加 `id="xxx"`，例如 `<div class="project-card" id="core">`
- **子级章节**：在子项的外层容器上加 `id="yyy"`，例如 `<div style="display: flex; gap: 12px; align-items: flex-start;" id="kahn">`

### 3. 添加侧边栏 HTML

在导航 `<nav>` 之后、`<div class="main">` 之前插入侧边栏：

```html
<aside class="sidebar">
    <div class="name">程愚瀚</div>
    <div class="title">Java 后端和 agent 开发</div>
    <nav>
        <a href="#section1" class="active"><span class="dot"></span> 章节一</a>
        <a href="#section2"><span class="dot"></span> 章节二</a>
        <div class="sidebar-sub-group">
            <a href="#sub1" class="sidebar-sub"><span class="dot"></span> 子项一</a>
            <a href="#sub2" class="sidebar-sub"><span class="dot"></span> 子项二</a>
        </div>
    </nav>
</aside>
```

- 顶级项用 `<a>` + 类 `active`
- 子项用 `<a>` + 类 `sidebar-sub`（自带缩进和小圆点）

### 4. 添加 CSS

在页面的 `<style>` 块中加入：

```css
.sidebar-sub { display: flex; align-items: center; gap: 8px; padding: 6px 14px; border-radius: 8px; font-size: 13px; font-weight: 500; color: #64748b; text-decoration: none; transition: all 0.2s; }
.sidebar-sub:hover { background: #f1f4f9; color: #0f172a; }
.sidebar-sub.active { background: #eef2ff; color: #4f46e5; }
.sidebar-sub .dot { width: 4px; height: 4px; border-radius: 50%; background: #cbd5e1; flex-shrink: 0; transition: background 0.2s; }
.sidebar-sub:hover .dot { background: #4f46e5; }
.sidebar-sub.active .dot { background: #4f46e5; }
.sidebar-sub-group { margin: 2px 0 6px 20px; border-left: 2px solid #e2e8f0; padding-left: 6px; }
```

### 5. 添加滚动高亮脚本

在页面底部（`</body>` 前）加入：

```html
<script src="../资源/js/main.js"></script>
<script>
    document.addEventListener('DOMContentLoaded', function() {
        function updateProjectNav() {
            var cards = document.querySelectorAll('.project-card[id], section[id]');
            var subTargets = document.querySelectorAll('[id="sub1"],[id="sub2"]');
            var navLinks = document.querySelectorAll('.sidebar nav a, .sidebar-sub');
            var current = '';
            cards.forEach(function(c) {
                var top = c.offsetTop - 130;
                var bottom = top + c.offsetHeight;
                if (window.scrollY >= top && window.scrollY < bottom) {
                    current = c.getAttribute('id');
                }
            });
            subTargets.forEach(function(s) {
                var top = s.offsetTop - 130;
                var bottom = top + s.offsetHeight;
                if (window.scrollY >= top && window.scrollY < bottom) {
                    current = s.getAttribute('id');
                }
            });
            navLinks.forEach(function(link) {
                link.classList.remove('active');
                if (link.getAttribute('href') === '#' + current) {
                    link.classList.add('active');
                }
            });
        }
        window.addEventListener('scroll', updateProjectNav);
        window.addEventListener('load', updateProjectNav);
        updateProjectNav();
    });
</script>
```

注意：`subTargets` 数组需要列出所有子项的 ID，例如 `'[id="kahn"],[id="rollback"],[id="visual"]'`。

### 6. 去掉 margin-left:0

如果主容器或 footer 有 `style="margin-left: 0"` 的内联样式，需要去掉，让 CSS 中的 `margin-left: 220px` 生效。
